# Zero BS CRM Improvements

A collection of lightweight WordPress plugins that extend and improve the experience of [Zero BS CRM (Jetpack CRM)](https://wordpress.org/plugins/zero-bs-crm/).

## Plugins

### [Add to Admin Menu](./add-to-admin-menu/)

Adds **"Contact"** and **"Company"** quick-add links to the WordPress admin bar's "New" dropdown for Zero BS CRM users.

- No JavaScript, no CSS, no database queries
- Respects Zero BS CRM user capabilities (`admin_zerobs_customers`)
- Provides filters and action hooks for customization
- Works on both admin and front-end toolbar

**Requirements:** WordPress 5.8+, PHP 7.4+, Zero BS CRM / Jetpack CRM

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- Zero BS CRM (Jetpack CRM) — [wordpress.org/plugins/zero-bs-crm](https://wordpress.org/plugins/zero-bs-crm/)
